## Setup this computer for Agent assistance

Make the following available all agents and all sessions to use:

- ~/Agents/AGENTS.md, copied from this repo if empty
- ~/Agents/skills, cloned from this repo

Reference `~/Agents/AGENTS.md` with instructions to read and follow it in:

- ~/.config/opencode/AGENTS.md
- ~/.claude/CLAUDE.md
- Wherever you save permanent Agent rules
